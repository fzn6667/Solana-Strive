# 🚀 Solana Strive: The Accountability Protocol

Solana Strive is a decentralized commitment protocol built for the Colosseum Frontier Hackathon. It solves the problem of self-discipline by allowing users to put their SOL on the line.

## ✨ Features
- **Staked Goals:** Lock SOL into a trustless smart contract.
- **Financial Incentive:** Funds are only returned if the goal is achieved.
- **Built on Solana:** Fast transactions and near-zero fees.

## 🛠️ Tech Stack
- **Smart Contract:** Rust & Anchor Framework
- **Deployment:** Solana Devnet
- **Testing:** TypeScript (Mocha/Chai)
